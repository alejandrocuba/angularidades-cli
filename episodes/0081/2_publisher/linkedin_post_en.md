Ready for Angular 22? This week the framework team will stabilize Signal Forms and Resources, but these aren't the only updates. To help you prepare for your migration, we invited Jorge Cano to review the key changes and technical details that v22.0.0 brings.

Topics we addressed during the conversation:
✔️ Signal Forms stabilization, native debounce() validation, and the transition away from RxJS for basic form logic.
✔️ The new @Service decorator for declaring clean singleton services, and asynchronous dependency injection via injectAsync.
✔️ SSR improvements with default incremental hydration and stream-based resource caching.
✔️ New experimental developer tools with MCP.
✔️ Support for Node 26 in the Angular CLI.
