# Contribution and Maintenance Guide

This repository handles "Prompts as Code". To maintain the quality of the AI outputs, follow these rules:

## Update Knowledge (Skills)
When the Angular team releases a new version, update the agent's context.
Do not modify the glossary manually. Run from the project root:
`npx skills update` (or replace the content of `.agents/skills/angular-developer.md` with the latest version from the official GitHub).

## Refactor Prompts
- If the AI generates generic or tutorial-style questions, tighten the constraints in `.agents/planner/system_prompt.md`.
- If you notice the AI using recurring marketing clichés, add the exact phrase to the blacklist in `.agents/publisher/system_prompt.md`.
- If the YouTube Chapters format fails, adjust the structure rules in `.agents/publisher/output_schema.md`.
- **Golden Rule:** Do not correct the same error generated by the AI in each episode. Fix the prompt once and commit it.
